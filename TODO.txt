https://www.imdb.com/interfaces/

1. Download title.basics.tsv.gz and collect all tconst where titleType=='movie' as Set[String].

2. Download title.ratings.tsv.gz and collect top 1000 tconst where tconst in Set of tconst found in 1a.

3. Iterate back through title.basics.tsv.gz with the list of tconsts and averageRatings from 2 and build the JSON:

{
	tconst: String,
	averageRating: Double,
	primaryTitle: String,
	startYear: Int,
	isAdult: Boolean,
	runtimeMinutes: Int,
	genres: List[String]
}

4a. Download title.principals.tsv.gz and build a map of tconst -> basic movie details.
4b. For each tconst in 4a, collect all nconst where category=='actor' or category=='actress' in title.principals.tsv.gz.
4c. Append to JSON object in step 3:

{
	actorsNconst: List[String], // Name IDs of actors in the film.
}

5a. Download name.basics.tsv.gz and map to JSON Object:

{
	nconst: String,
	primaryName: String,
	birthYear: Int,
	deathYear: Int
}

5b. Map the nconsts: List[String] in step 4c to a List of actor/actress JSON objects built in 5a.
